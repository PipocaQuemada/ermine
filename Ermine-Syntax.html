(#) :: AReview s t a b -> b -> t

An infix alias for review. 

 unto f # x ≡ f x
 l # x ≡ x ^. re l

This is commonly used when using a Prism as a smart constructor. 

>>> _Left # 4
Left 4

But it can be used for any Prism 

>>> base 16 # 123
"7b"
